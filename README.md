# Project 4 — Airflow Orchestration

Production-style workflow orchestration project using Apache Airflow.

## Goal
This project orchestrates the mini data platform built across previous projects:
- Project 1: Batch ETL
- Project 2: Analytics API
- Project 3: Streaming pipeline
- Project 4: Orchestration with Airflow

## Local Setup
- Apache Airflow
- Docker Compose
- Postgres metadata database
